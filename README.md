# Resource Pals

A top down resource gathering multiplayer shooter. 

## Getting Started

To play the game go to www.resourcepals.com. To install locally

```
  mkdir teampals
  cd teampals
  git clone https://github.com/bjadboys/teampals.git

```

### Prerequisites

We're using node 8.6. Not sure how far back in node will work. 

```
https://nodejs.org/en/
```

### Installing

To run the project:

```
yarn
yarn start-dev
```

to go: localhost:8080

```
until finished
```

End with an example of getting some data out of the system or using it for a little demo

## Running the tests

Tests store and server.
```
yarn run test
```
